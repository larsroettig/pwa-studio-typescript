

Install TypeScript Setup
```bash
yarn add --dev typescript @babel/preset-typescript @babel/plugin-proposal-optional-chaining @babel/plugin-proposal-nullish-coalescing-operator @types/graphql @typescript-eslint/eslint-plugin@^5.9.1 @typescript-eslint/parser@^5.9.1 eslint-config-prettier @types/node @types/react-router-dom
```

Testing Stack:
``` 
yarn add --dev husky @types/jest @testing-library/react @testing-library/jest-dom jest ts-jest concurrently
``
